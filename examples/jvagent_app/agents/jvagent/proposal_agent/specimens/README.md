# Proposal Specimen Corpus

This directory contains specimen proposals, a structural template, and a writing guide used by the `proposal_draft` skill to generate high-quality drafts.

## Structure

- `template.md` — The canonical proposal structure skeleton
- `guide.md` — Writing principles and tone guidance
- `retail/` — Specimen proposals for retail/e-commerce clients
- `enterprise/` — Specimen proposals for enterprise/enterprise SaaS clients

## Tags

| Directory | Tags |
|-----------|------|
| `retail/` | retail, e-commerce, mobile, consumer, B2C |
| `enterprise/` | enterprise, SaaS, B2B, data-platform, integration |

## Adding Specimens

Add `.md` files to the appropriate subdirectory. Each file should represent a real or anonymized past proposal. Keep the format clean Markdown with clear section headings matching the template structure.

Update this README when adding new specimen categories.
